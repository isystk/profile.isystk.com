Nuxt.js TypeScript Vuetify サンプル
====

Nuxt.js、TypeScript、Vuetify を利用したサンプルです。

## Description

Nuxt.js、TypeScript、Vuetify を学習する目的で、自分のポートフォリオを作成してみました。

## Requirement

* Nuxt.js@2.10.2
* Vue.js@2.6.10 
* Vuetify@2.1.6 

## Usage


## Install

``` bash

# yarn(パッケージマネージャ)のインストール
$ npm install -g yarn

# 依存モジュールのインストール
$ yarn install

# ビルド&起動
$ yarn run dev
$ curl http://localhost:3000/
```

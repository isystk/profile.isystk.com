## Description

📗 isystk のポートフォリオです。Typescript&Nuxt.jsで最近のフロントエンドの技術を取り入れて作成しました。

### ディレクトリ構造
```
.
├── src (Nuxt関連のソースファイル)
│    ├── @types
│    ├── assets
│    ├── common
│    ├── components
│    ├── interfaces
│    ├── middleware
│    ├── pages
│    ├── plugins
│    ├── static
│    ├── store
│    ├── test
│    └── utilities
├── .babelrc (Babel設定ファイル)
├── .vscode (VSCode設定ファイル)
├── jest.config.js
├── nuxt.config.ts
├── package.json
└── tsconfig.json
```

## 🖊️ VS. 

### 利用している技術
- Nuxtjs 2.10
- Typescript
